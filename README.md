# Simple CRUD Console Application in C#

A simple console-based CRUD (Create, Read, Update, Delete) application written in C# for managing a list of people (like a basic phonebook or contact manager).

This project is built as a beginner-friendly example to demonstrate basic C# concepts such as:
- Structs
- Lists
- Console input/output
- Menu-driven programs
- Search, edit and delete operations

## Features

- ➕ **Add** a new person (Name + Number)
- ✏️ **Edit** an existing person by name
- 🔍 **Search** for a person by name
- 🗑️ **Delete** a person (with confirmation)
- 📋 **List** all people with numbering
- Clean console menu interface
